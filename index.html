Laços de Repetição (Loops)

São estruturas que permitem executar um bloco de código várias vezes, com base em uma condição específica. São fundamentais para automatizar tarefas repetitivas. Os principais tipos incluem: ￼ • for: Usado quando se sabe antecipadamente o número de repetições. • while: Executa enquanto uma condição for verdadeira. • do...while: Semelhante ao while, mas garante que o bloco seja executado pelo menos uma vez. Esses laços são comuns em linguagens como JavaScript, Java, Python e C# .

Linguagens de Programação

São conjuntos de regras e sintaxes que permitem aos desenvolvedores escrever instruções que os computadores podem entender e executar. Exemplos populares incluem: • JavaScript: Usada principalmente para desenvolvimento web. • Python: Conhecida por sua simplicidade e legibilidade. • Java: Amplamente utilizada em aplicações corporativas. • C#: Desenvolvida pela Microsoft, comum em aplicações Windows. • C++: Usada em sistemas que requerem alto desempenho. Principais Tags Usadas No contexto do desenvolvimento web, especialmente em HTML e CSS, algumas tags e propriedades são amplamente utilizadas: • HTML: • -p>/: Define um parágrafo. • -h1> a -h6>: Definem títulos, sendo -h1> o mais importante. • -div>: Usada como contêiner genérico para agrupar elementos. • -span>: Usada para agrupar texto inline para estilização. • -a>: Define um hiperlink. • -img>: Incorpora imagens. • CSS: • color: Define a cor do texto. • background-color: Define a cor de fundo de um elemento. • font-size: Define o tamanho da fonte. • text-align: Alinha o texto horizontalmente. • text-transform: Controla a capitalização do texto.

Propriedades de Estilização CSS
background-color /p> Define a cor de fundo de um elemento HTML. Pode ser especificada por nome (e.g., red), código hexadecimal (e.g., #FF0000) ou valores RGB (e.g., rgb(255, 0, 0)). font-size

Controla o tamanho do texto. Pode ser definido em unidades como pixels (px), em (em), rem (rem), porcentagem (%), entre outras. text-align

Alinha o texto dentro de um elemento. Os valores comuns incluem: • left: Alinha à esquerda (padrão). • right: Alinha à direita. • center: Centraliza o texto. • justify: Justifica o texto, alinhando ambos os lados.

text-transform

Controla a capitalização do texto. Os valores incluem: • uppercase: Converte o texto para letras maiúsculas. • lowercase: Converte o texto para letras minúsculas. • capitalize: Capitaliza a primeira letra de cada palavra. • none: Mantém o texto como está.

Console

O console é uma interface que permite a interação entre o usuário e o sistema ou aplicação por meio de texto. Em programação, é frequentemente usado para: • Depuração: Exibir mensagens que ajudam a identificar e corrigir erros. • Entrada/Saída: Receber dados do usuário e exibir resultados. • Execução de Comandos: Executar scripts ou comandos diretamente. Por exemplo, em JavaScript, console.log("Mensagem") imprime “Mensagem” no console do navegador.
